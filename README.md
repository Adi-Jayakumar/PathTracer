# PathTracer
A path tracer written in C++ that executes on the CPU, parallelised using OpenMP, and GPU accelerated using CUDA - inspired by vchizov's smallpt explained.

# Primitives
 - Triangles
 - Planes (infinite)
 - Spheres

# Material types
 - Diffuse
 - Specular
 - Refractive
 - Glossy Specular
 - Glossy Refractive
